Prework Study Guide

HTML

  • The head element contains info about the webpage.
  • The body element represents the visible content shown to the user.
  • The section element is used to add ids to a webpage.
  • The footer element end the code and represents what is seen at the bottom of a page

CSS

  • CSS describes how HTML elements are displayed on a screen or webpage
  • CSS stands for Cascading Style Sheet and is often refered to as the style of code
  • CSS will be gone over in futer lessons

Git

  • Git or Git Bash is different from GitHub in that it uses a computer terminal to execute tasks
  • Git comands are used in the command line to interact with code
  • Git status: checks what branch we are currently on
  • Git checkout -b branch-name: creates a new branch and switches to it
  • Git comands are also used to save and transfer code or code branches to a server like GitHub
  • Git has a variety of differnt comands like git status, git push, and git pull

JavaScript

  • JavaScript is used to interact with a server more directly
  • JavaScript often is refered to as the accent of a code
  • JavaScript allows you to change the shape of what your code will look like
